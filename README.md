# 議員SNS

議員SNSは、地方自治体議員のSNS情報をまとめて知ることができるサービスです！  
http://giinsns.com/
  
  
現在は、品川区数名、東京都1名のTwitter情報のみですが、東京都と23区議員のTwitterとFaceBook情報を掲載する予定です。
